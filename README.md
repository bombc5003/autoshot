# 오토샷 (autoshot)

화면 자동기록 · 캡처 자동축적 · PDF 자동병합 도구
Windows 10 / 11 (64bit) 단일 실행본

---

## 무엇을 하나

오토샷은 일정 주기로 화면을 캡처하여 자동기록함에 누적 저장한 뒤, 일정 주기로 모아 PDF 1개로 자동 병합하는 도구이다.
모든 캡처에는 시각·일련번호·자동/수동 구분·사용자 라벨 캡션이 상단 띠로 합성되며,
하단 푸터에는 빌드 식별자가 각인되어 출력물의 원본성을 사후 확인할 수 있다.

폐쇄망 · 오프라인 동작.
외부 통신 경로 없음.

---

## 권장 용도 (BRAND_CAPTIONS 20슬롯)

소송 증거 보존, 행정심판 자료 축적, 온라인 거래 분쟁 증거, 원격근무 작업 기록,
강의 스틸컷 제작, 영상 감상 노트, 교육 자료 제작, 화상회의 캡처,
실험 일지, 전산 오류 재현, 근무시간 입증, 저작권 침해 기록 등.

---

## 주요 기능

자동 촬영 — 사용자 지정 주기(기본 15초)로 PNG 누적 저장
수동 촬영 — 단축 버튼으로 즉시 캡처
자동 병합 — 사용자 지정 주기(기본 30분)로 누적 PNG → PDF 1개 병합
캡션 자동 합성 — 상단 28px 띠(시각·일련·라벨), 하단 24px 푸터(빌드 식별자)
사용자 라벨 — 캡션 띠에 사용자 정의 문구 부착
폴더 자동 분리 — 자동/수동 캡처 별도 분기
설정 영구 보존 — `오토샷_config.json`

## 무결성 검증

다운로드한 `autoshot.exe`의 SHA-256 해시를 `SHA256SUMS.txt` 와 대조.

PowerShell 1줄 명령

```
Get-FileHash autoshot.exe -Algorithm SHA256
```

출력된 해시가 SHA256SUMS.txt 의 값과 일치하면 원본 무변조.
